Definitions & Concepts

HTML : "Hyper Text Markup Language"

> behind every website is a HTML-Code describing the website

> it's a standart programming-language for websites

> it's responsible for the structure of a websites

> Markup makes the code readable for the omputer

> Hyper-Text creates links to another website

CSS: "Cascading Style Sheet"

> CSS is what makes a website beautiful

> it's responsible for the Design and Layout

> it's connected with html-data

HTPP : "Hyper Text Transfer Protocols"

> language for communication between computers

> HTTP: exchange of informatiions in a simple form of text / HTTPS: data gets transmitted encrypted

> it's in use if we search for sth. in the browser

   > if we click on a link, insert an URL it's releasing a GET request

   > request contains the host-name and the needet ressource

> with HTTP you're also able to send informations (f.e. in a login formular where you insert email and password)

common status codes: 200 (request in process) 301(Moved Permanently) 404(Page Not Found)

JavaScript = a more complicated language than HTML but it makes interactions possible

JSON = a language that translates between the human programming language and the computer language

TLS = Transport Layer Security - a widely adopted security protocol that is encrypting the communication between web applications and servers

DNS = the Domain Name System turns domain names into IP addresses, wich browsers use to load internet pages
