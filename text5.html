JAVA

It's a general purpose, object oriented programming language intendet to let programmers write once, run everywhere. That means the code can run on all platforms that support Java without the need to recopile.

Learning curve: It's easy to learn cause it has a reduced scope of language. It would take around a year to learn Java.

Use: web applications, apps, java-desktop application --> for example: Spotify, Google, Netflix

Syntax: The syntax of Java is similar to C&C++ but has fewer low-level facilities than either of them. Java contains objects, classes, methodes, instance variables and more.

Performance: Java was considered slower than C&C++
